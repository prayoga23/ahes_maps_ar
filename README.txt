Web AR Navigation Template
============================

How to use:
1. Host these files on a secure (HTTPS) server or GitHub Pages.
2. Replace 'arrow.glb' with your own 3D arrow model (same name or change in app.js).
3. Open the page with parameters, e.g.:
   https://yourdomain.com/index.html?lat=-7.288102&lng=112.783574

Make sure location and camera permissions are enabled on the device.
